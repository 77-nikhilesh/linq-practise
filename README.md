# linq-practise
A personal practice repository for learning and experimenting with LINQ queries in C#.

## Methods Practiced

### Filtering Methods
* .Where()
* .OfType()

### Partitioning Methods
* .Skip()
* .Take()
* .SkipLast()
* .TakeLast()
* .SkipWhile()
* .TakeWhile()

### Projection Methods
* .Select()
* .SelectMany()

### Casting and Conversion Methods
* .Cast()

### Division / Grouping Methods
* .Chunk()
